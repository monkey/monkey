Monkey + PHP

Före du börjar med konfigurering och installation av PHP, förutsätter jag att du har Monkey installerat på ditt system.

Det spelar ingen roll om du har PHP installerat med Apache eller någon annan httpd, eftersom du kan installera PHP i en annan katalog, det funkar utmärkt ändå.

  • Ladda ner senaste versionen av PHP:
      På den officiella hemsidan http://www.php.net kan du hitta den senaste versionen av denna utmärkta hypertext förbearbetare.
  • Installation av PHP för Monkey:
      Från kommandoraden :

        # tar zxfv php-4.x.x.tar.gz
        # cd php-4.x.x
        # ./configure --prefix=/home/my_home/php-4.x.x
        # make
        # make install

      Då kommer PHP installeras i din hemkatalog. Om du vill ha stöd för MySQL,
      måste du efter ./configure lägga till --with-mysql eller någonting annat du kan tänkas vilja aktivera
  • Lägga till PHP-bibliotek till systemet :
      Nu måste vi ha en kopia av konfigurationsfilen för PHP, som finns i samma katalog som källkoden, med målnamnet 'php.ini' där vi installerat PHP:

        # cp php.ini-dist /home/my_home/php-4.x.x/lib/php.ini
      Följande steg måste utföras som root. Lägg till följande rad i slutet på "/etc/ld.so.conf" :

        /home/my_home/php-4.x.x/lib

      Efter det, kör följande kommando :

        # ldconfig
  • Konfigurera om monkey.conf :
      Nu när vi har PHP installerat, måste vi ändra och konfigurera om filen monkey.conf (conf/monkey.conf)

      I filen måste vi definiera mime-typ och interpreter att bearbeta PHP-filerna med, så du måste lägga till följande rader:

        AddScript application/x-httpd-php /home/my_home/php-4.x.x/bin/php php

      Med detta definierat, kan du nu bearbeta filer med filändelsen .php. Om du du vill lägga till fler filändelser så kan du lägga till fler rader som ovan med fler filändelser
