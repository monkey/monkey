Monkey + PHP

Avant de vous donner les instructions pour configurer et installer PHP nous considéreront que Monkey est déja installé sur votre système.

Il n'est pas important que vous ayez déja installé PHP avec Apache ou un atre serveur HTTP. Installer PHP dans un autre répertoire ne posera pas de problèmes de compatibilité.

  • Téléchargez la dernière version de PHP :
      Sur le site officiel  http://www-php.net vous pourrez trouver la dernière version de ce grand préprocesseur hypertexte.
  • Installer PHP pour Monkey


    • Depuis un shell :

    # cd php-4.x.x
    # ./configure --prefix=/home/my_home/php/
    # make
    # make install

    Ceci installera PHP dans votre répertoire personnel. Si vous voulez ajouter le support  MySQL ajoutez --with-mysql à la ligne de commande de configure ainsi que toute autre option que vous souhaitez.

  • Ajouter les librairies PHP au système :

    • Maintenant noius devons copier le fichier de configuration de PHP (php.ini-dist)
    qui se trouve dans le répertoire source de Monkey vers le répertoire ou nous venons
    d'installer PHP en le renommant "php.ini":

      # cp php.ini-dist /home/my_home/php/lib/php.ini
      Ceci doit êtr exécuté en root. Ajoutez cette ligne à la fin de "/etc/ld.so.conf" :

        /home/my_home/php/lib

      Ensuite exécutez la commande:

        # ldconfig
  • Reconfigurer monkey.conf :

    • Après avoir installé PHP correctement, vous devez éditer et reconfigurer le fichier monkey.conf (conf/monkey.conf).n the file, vous devez définir le type mime et l'interprèteur pour exécuter les fichiers PHP en ajoutant cette ligne :

      AddScript application/x-httpd-php /home/my_home/php/bin/php php
    Vous pouvez maintenant traiter les fichier .php. Si vous souhaiter utiliser PHP pour traiter d'autres extensions vous devez ajouter d'autre lignes avec l'extension que vous souhaitez.
